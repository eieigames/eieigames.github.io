# Eiei Games website

Static GitHub Pages website for [eieigames.com](https://eieigames.com/).

## Search setup

The repository includes:

- `robots.txt`, which allows crawling and points crawlers to the sitemap.
- `sitemap.xml`, which lists the canonical public pages.
- Unique page titles and meta descriptions.
- Canonical URLs for the preferred `https://eieigames.com` domain.
- Crawlable internal navigation between all public pages.
- `WebSite` structured data on the home page to identify the site name.
